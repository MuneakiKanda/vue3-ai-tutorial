# Vue3 AI チュートリアル

AIを活用して「人間向けの完璧なVue3チュートリアル」を生成するためのプロンプト集です。

## 概要

このリポジトリには、汎用LLM（ChatGPT / Claude 等）に読み込ませることで、体系的かつ実践的なVue3チュートリアルを生成させるための **指示書（プロンプト）** が含まれています。

## 対象バージョン（固定）

| パッケージ | バージョン |
|---|---|
| Vue | 3.5.29 |
| Vite | 6.x |
| Vue Router | 4.x |
| Pinia | 2.x |
| TypeScript | 5.x |
| Node.js | 20.x LTS 以上 |

> **注意**: Vue 3.6.0（Vapor Mode）はまだ beta のため、本チュートリアルでは安定版 3.5.x に限定しています。

## ファイル構成

```
vue3-ai-tutorial/
├── README.md              ← このファイル（プロジェクト概要）
├── .gitignore
├── PROMPT.md              ← メイン指示書（AIに最初に読み込ませる）
├── CHAPTER_TEMPLATE.md    ← 章テンプレート（各章の出力フォーマット）
├── CURRICULUM.md          ← カリキュラム（全20章の目次と概要）
└── tutorial/
    └── v3.5/              ← Vue 3.5.x 対象のチュートリアル本文
        ├── chapter01.md
        ├── chapter02.md
        ├── chapter03.md
        └── ...            （全20章を順次追加）
```

## 使い方

### 1. メイン指示書を投入

`PROMPT.md` の内容をそのままLLMのチャットに貼り付けます。これによりAIは「Vue3チュートリアル専門のテクニカルライター」として振る舞います。

### 2. 章テンプレートを投入

続けて `CHAPTER_TEMPLATE.md` を貼り付けます。AIが各章を生成する際の出力フォーマットが定義されます。

### 3. カリキュラムを投入

`CURRICULUM.md` を貼り付けて全体の章構成を共有します。

### 4. 章ごとに生成を依頼

以下のように依頼します：

```
第1章を CHAPTER_TEMPLATE.md のフォーマットに従って生成してください。
```

1章ずつ順番に生成し、品質を確認しながら進めることを推奨します。

## チュートリアルの特徴

- **Composition API + `<script setup>` に統一** — Vue3推奨のモダンスタイルに集中
- **TypeScript 前提** — 型安全なコードでベストプラクティスを最初から学ぶ
- **段階的サンプル積み上げ** — 各章で小さなサンプルコードを作りながら学ぶ実践形式
- **完全動作するコード** — コード断片ではなく、必ず動作する完全なコードを提示
- **「なぜそう書くのか」を重視** — 暗記ではなく理解を促す説明

## 対象読者

- HTML / CSS / JavaScript の基礎は理解している
- フロントエンドフレームワークの経験はない（またはほぼない）
- TypeScript は学習中〜基礎レベル
